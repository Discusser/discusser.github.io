This is a script made with bash to automatically backup certain folders to a backup folder and to an external hard drive if it is plugged in.

BackupScript.sh


#!/bin/bash

# Schedule with anacron

home=/home/discusser
documents=$home/Documents
backup_location=$documents/Backups
allBackups=($documents/School $documents/Scripts)

lsusb | grep -q "Western Digital Technologies, Inc. Elements Portable"
if [ $? == 0 ]
then
	for i in ${allBackups[@]}
	do
		cp -r $i /media/discusser/External Hard Drive/Backups
	done
else
	for i in ${allBackups[@]}
	do
		cp -r $i $backup_location
		chmod -R 774 $backup_location/$(echo $i | sed "s/.*\///")
		chown -R discusser:root $backup_location/$(echo $i | sed "s/.*\///")
	done
fi

How to use:

First of all, I want to point out a few things. File paths will not be the same for everyone, this is just my version of the script on Debian 11. The following can be personalised: home, documents, backup_location, allBackups, the external hard drive name, the external hard drive path, and discusser:root at the chown command (I recommend that you only change discusser to your own username)

1. Optional: if you don't have an external hard drive, you can remove line 11 to 18 and line 25.

2. Configure file paths so that the script works with your setup (don't forget to change the external hard drive name).

3. Change discusser:root next to chown to your_username:root (type man chown in your terminal for more information).

4. Open /etc/anacrontab with a text editor.

5. Add an entry on a new line, the format is period (how frequently should the job run, in days), delay (how much time should there be between the anacron program starting and the job being run, job identifier (a unique name for your job), and command (what command it should run). If you wanted to run your command each day, 5 minutes after the anacron program starts (it starts when the computer boots up), the entry would look something like this: 1 5 backup-script /path/to/script.sh

6. Save the file and exit your text editor.

If you did everything correctly, your files should be scheduled to backup in a day.

Note: anacron is made for devices that aren't always turned on, so it counts even when the computer is turned off by comparing current time to whenever you turned off the computer.
