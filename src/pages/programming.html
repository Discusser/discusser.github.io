File hosting via GitHub

While this might sound stupid at first (and will probably sound more stupid once I explain it), this isn't what you think it is.

One day I opened my ShareX screenshots folder, only to see that it had grown to multiple gigabytes in size. This was unacceptable for me and I decided I wanted to host my pictures externally. Not wanting to spend money, I got the idea to use GitHub to store my images.

Now, whenever I record something or take a screenshot using ShareX, a python script runs and uploads the image or video to GitHub. In order to not have to log in every time I wanted to upload something, I simply set up an SSH key on my computer that keeps me logged in at all times.

For convenience, I created a (badly made) file browser that lets me browse that repository. To access repositories on GitHub without going through the website, you need to make a token. Fortunately, you can configure token permissions, so don't even try doing anything with it.
