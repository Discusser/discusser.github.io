Discusser's Website - Minecraft sound extractor

Installation

Download the latest release from GitHub.

This requires Python 3 or above to be installed on your computer.

How to use

Locate your .minecraft folder, you can usually find it in %appdata%\.minecraft\ on Windows. Place the python file in there and run it, then choose your version. Once the script has finished running, the extracted sounds will be in .minecraft\extractedSounds\.
